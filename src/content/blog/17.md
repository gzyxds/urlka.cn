## 📱 套餐详情页面介绍

在 172 号卡平台，每个套餐都有独立的详情页面，帮助用户快速了解产品信息。

### 页面主要模块

| 模块 | 说明 |
|------|------|
| **优惠详情** | 套餐资费、流量、通话等核心信息 |
| **快递方式** | 发货快递公司及运费说明 |
| **激活方式** | 运营商及激活渠道指引 |
| **充值链接** | 首充话费官方渠道 |
| **办卡年龄** | 申请年龄限制要求 |
| **合约期** | 套餐合约期限说明 |
| **不发货地区** | 无法配送的区域列表 |
| **定向范围** | 定向流量适用 APP |
| **激活流程图** | 详细激活步骤图解 |

---

## 🔍 如何查看套餐详情

### 方式一：产品链接直达

代理或用户可通过产品专属链接直接访问详情页：

```
https://172.lot-ml.com/h5orderEn/index?pudID=产品 ID&userid=代理 ID
```

### 方式二：店铺页面选择

1. 打开 172 号卡店铺页面
2. 浏览产品列表
3. 点击感兴趣的产品
4. 进入详情页查看完整信息

## 📋 关键信息解读

### 1️⃣ 优惠详情

- **月租资费**：套餐每月费用
- **流量额度**：通用流量 + 定向流量
- **通话时长**：免费通话分钟数
- **优惠期**：优惠资费持续时间

### 2️⃣ 快递方式

| 运营商 | 快递 | 运费 |
|--------|------|------|
| 中国移动 | 京东/顺丰 | 包邮 |
| 中国联通 | 京东/顺丰 | 包邮 |
| 中国电信 | 京东/顺丰 | 包邮 |
| 中国广电 | 京东/顺丰 | 包邮 |

### 3️⃣ 激活方式

- **中国移动**：微信公众号/中国移动 APP
- **中国联通**：联通手厅 APP/小程序
- **中国电信**：电信营业厅 APP
- **中国广电**：广电 APP

### 4️⃣ 充值链接

详情页提供官方充值入口：
- 充值金额：通常 50 元/100 元起
- 充值渠道：运营商官方渠道
- 到账时间：24 小时内

### 5️⃣ 办卡年龄

| 运营商 | 年龄限制 |
|--------|----------|
| 中国移动 | 16-60 周岁 |
| 中国联通 | 16-60 周岁 |
| 中国电信 | 16-60 周岁 |
| 中国广电 | 16-65 周岁 |

### 6️⃣ 合约期

- **无合约**：随时可注销
- **短期合约**：3-6 个月
- **长期合约**：12-24 个月

### 7️⃣ 不发货地区

常见不发货区域：
- 新疆、西藏等偏远地区
- 云南德宏、西双版纳
- 广西百色、崇左
- 具体以页面显示为准

### 8️⃣ 定向范围

定向流量适用 APP（示例）：
- 抖音、快手、腾讯视频
- 爱奇艺、优酷、哔哩哔哩
- 百度系、网易系、头条系

### 9️⃣ 激活流程图

页面提供详细激活步骤图解，包含：
- 实名认证流程
- 人脸识别步骤
- 首充操作指引

## 💡 使用技巧

### 给用户的建议

1. **仔细阅读优惠详情**，确认套餐符合需求
2. **查看不发货地区**，避免下单后无法配送
3. **了解合约期**，确认能否接受
4. **保存激活流程图**，方便后续操作

### 给代理的建议

1. **熟悉产品详情**，便于解答用户咨询
2. **生成宣传海报**，页面支持一键生成
3. **分享专属链接**，订单自动关联
4. **关注更新信息**，套餐信息可能调整

## ❓ 常见问题

### Q1: 套餐信息与实际不符怎么办？

套餐信息以运营商官方为准，如有调整会及时更新页面。

### Q2: 如何生成产品宣传海报？

详情页提供"生成海报"功能，点击即可生成专属推广海报。

### Q3: 订单在哪里查询？

下单后可在订单查询页面输入手机号查询物流和激活状态。
